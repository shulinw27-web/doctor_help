# 医生帮（Doctor Help）

院内本地部署的 AI 医生助手演示项目，用于帮助医生汇总检验、影像和历史就诊资料，形成可追溯的资料预读与异常提示。

医生帮用于支持临床工作，但不替代医生诊断，也不替代医院现有 HIS、EMR、LIS、PACS/RIS 等业务系统。当前版本仅使用模拟数据，严禁用于真实患者数据处理或临床诊疗。

## 项目结构

- `doctor-help-frontend`：Electron + Vue 3 + TypeScript 前端项目。
- `doctor-help-backend`：Java 21 + Spring Boot 后端项目。
- `docs`：架构、接口和开发说明。
- `outputs`：演示推广材料。
- `医生帮项目对话纪要.md`：需求、决策、进度和恢复开发说明。

前端与后端可独立启动、构建和部署，仅通过 HTTP API 通信。

## 当前阶段

第一阶段为离线模拟演示：包含医生工作台、患者资料预读、知识库、管理驾驶舱、人员管理和数据接入状态展示。

生产试点将遵循只读优先原则：统一数据视图和 AI 辅助分析，正式业务操作仍在医院原有系统中完成。
